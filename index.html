💾 Backup & Restore
⬇️ Save Full Backup EXPORT
Downloads two CSV files to your computer — one for your expense transactions, one for income, budgets, goals, loans and savings. Use to back up before switching devices.
expensio_expenses_DATE.csv expensio_finance_DATE.csv
🔄 Restore from Backup REPLACE
Load a backup file saved by Expensio. Expenses CSV replaces your transaction list. Finance CSV restores goals, loans, income and savings. The app detects which file type automatically.
⚠ Replaces existing data — use only with a file previously exported by Expensio
📥 Import New Data MERGE
Import expenses from any CSV or Excel file — a bank statement, another app's export, or a custom spreadsheet. Columns are mapped automatically. You can choose to merge with existing data or replace it.
Accepts any file with: date, amount, category, note, method columns (in any order)
⚠️ Danger Zone
Each option is independent — clearing expenses does not touch your goals or loans, and vice versa.
🧾 Clear Expenses Only
Deletes all transaction history. Income, budgets, goals, loans & savings are kept.
💰 Reset Finance Data
Resets income, budgets, savings goals, loans & savings balance. Expenses are kept.
☢ Clear Everything
Permanently deletes ALL data — expenses, goals, loans, savings, income. Cannot be undone.
🔒 Privacy
Expensio is fully local. No expense data ever leaves your device except the anonymised summary sent to the AI provider when you generate insights.
